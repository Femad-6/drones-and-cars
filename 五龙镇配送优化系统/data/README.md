# 数据文件说明

本目录包含五龙镇配送优化系统所需的数据文件。

## 必需数据文件

### 1. 物流车配送距离.xlsx

**用途**: 车辆行驶距离矩阵，包含各地点间的实际道路距离。

**格式要求**:

- 第一列: 起点名称（字符串）
- 其他列: 目的地名称（字符串）
- 单元格: 距离数值（公里，浮点数）

**示例格式**:

| Unnamed: 0 | 配送中心 | 五龙镇 | 丰峪村 | ... |
|------------|----------|--------|--------|-----|
| 配送中心   | 0.0      | 15.2   | 18.7   | ... |
| 五龙镇     | 15.2     | 0.0    | 8.3    | ... |
| 丰峪村     | 18.7     | 8.3    | 0.0    | ... |

### 2. 五龙镇各村距离矩阵.xlsx

**用途**: 村庄间的距离数据，主要用于无人机路径规划。

**格式要求**: 类似物流车距离矩阵，但主要包含村庄间的距离。

## 数据来源

1. **实地测量**: 通过GPS设备或地图软件测量实际距离
2. **地图数据**: 使用百度地图、高德地图等获取路径距离
3. **历史数据**: 基于过往配送记录的统计数据

## 数据质量要求

1. **准确性**: 距离数据应尽可能接近实际值
2. **完整性**: 覆盖所有配送点之间的距离
3. **一致性**: 距离矩阵应满足三角不等式
4. **时效性**: 数据应定期更新以反映道路条件变化

## 注意事项

1. 车辆距离和直线距离的区别：
   - 车辆距离：实际道路行驶距离，受道路条件影响
   - 直线距离：两点间最短距离，用于无人机飞行

2. 距离单位：统一使用公里(km)

3. 数据更新：建议每季度更新一次距离数据

## 自定义数据

如果需要使用其他地区的数据，请：

1. 按照上述格式准备距离矩阵文件
2. 在配置文件中更新位置坐标
3. 修改需求量数据
4. 调整车辆和无人机参数
