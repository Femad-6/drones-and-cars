# 智能距离计算功能使用指南

## 🎯 功能概述

智能距离计算功能通过集成百度地图API，能够：

- 🚗 **车辆距离**：获取真实的行驶距离和路径
- 🚁 **无人机距离**：计算两点间的直线距离
- 📍 **地点解析**：自动将地址转换为坐标
- 🛣️ **智能矩阵**：生成准确的距离矩阵用于优化

## 🔑 百度地图API申请

### 1. 注册百度开发者账号

- 访问：<https://lbsyun.baidu.com/>
- 注册并完成实名认证

### 2. 创建应用

- 进入控制台 → 应用管理 → 我的应用
- 点击"创建应用"
- 应用类型：选择"服务端"
- IP白名单：填写您的服务器IP（开发时可填写0.0.0.0/0）

### 3. 获取API密钥

- 创建成功后，在应用列表中找到您的应用
- 复制"访问应用(AK)"，这就是您的API密钥

### 4. 配额说明

- 新用户每日免费配额：
  - 地理编码：6,000次/天
  - 路径规划：3,000次/天
- 超出免费配额后按次计费

## 🚀 使用步骤

### 步骤1：选择距离计算方式

在Web应用的"智能距离配置"区域：

- 选择"智能API距离计算"
- 或选择"使用默认数据"（使用内置距离）

### 步骤2：配置API密钥

- 在"百度地图API密钥"输入框中输入您的AK
- 点击"🔍 验证API"按钮确认密钥有效

### 步骤3：配置配送地点

可以选择两种方式：

#### 方式A：使用默认地点

- 选择"使用默认地点"
- 系统将使用内置的五龙镇地点数据

#### 方式B：自定义地点列表

- 选择"自定义地点列表"
- 输入城市名称（如：济源市）
- 在文本框中每行输入一个地点名称，例如：

  ```
  五龙镇政府
  丰峪村委会
  上庄村
  文峪村
  济源市体育中心
  ```

- 点击"🌍 解析地点坐标"

### 步骤4：生成距离矩阵

- 确认API密钥有效且地点配置完成
- 点击"🚗 生成智能距离矩阵"
- 等待系统调用API获取距离数据

### 步骤5：保存和使用

- 生成完成后可点击"💾 保存距离矩阵"
- 在优化算法中将自动使用新的距离数据

## 📊 距离计算说明

### 车辆距离（百度地图API）

- **数据来源**：百度地图路径规划API
- **计算方式**：实际道路行驶距离
- **准确性**：考虑道路限制、交通规则
- **适用场景**：车辆配送路径优化

### 无人机距离（坐标计算）

- **数据来源**：Haversine公式计算
- **计算方式**：球面两点间最短距离
- **准确性**：理论直线距离
- **适用场景**：无人机飞行路径优化

## ⚠️ 注意事项

### API使用限制

- 注意每日免费配额限制
- 避免短时间内大量请求
- 系统已内置请求间隔控制

### 地点名称规范

- 使用准确的地点名称（如：XX村委会、XX政府）
- 包含具体的行政区划信息
- 避免使用过于简化的名称

### 网络要求

- 确保服务器能访问百度地图API
- 检查防火墙和代理设置
- 建议在网络状况良好时进行批量生成

### 数据质量

- API生成的距离比坐标计算更准确
- 适合对精度要求较高的应用场景
- 建议定期更新距离数据

## 🔧 故障排除

### API密钥验证失败

1. 检查密钥是否正确复制
2. 确认应用状态是否正常
3. 检查IP白名单配置
4. 验证配额是否充足

### 地点解析失败

1. 检查地点名称是否准确
2. 尝试添加更多地理信息
3. 确认城市名称设置正确
4. 检查网络连接状态

### 距离矩阵生成失败

1. 验证API密钥状态
2. 检查地点数据完整性
3. 确认网络连接稳定
4. 查看系统日志获取详细错误

## 💡 最佳实践

### 1. 测试先行

- 先用少量地点测试API功能
- 确认系统正常后再批量处理

### 2. 合理规划

- 根据配额合理安排生成时间

### 4. 监控使用

- 合理控制请求频率
