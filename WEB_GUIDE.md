# 五龙镇配送优化系统 - Web应用使用指南

## 🚀 快速启动

### 方法1：使用启动脚本（推荐）

```bash
# Windows用户
双击运行 start_webapp.bat

# 或者命令行运行
python start_webapp.py
```

```bash
streamlit run webapp.py
```

- Python 3.7 或更高版本
- 已安装requirements.txt中的依赖包

```bash
pip install -r requirements.txt
```

## 🔧 常见问题

### Q: 文件上传失败（AxiosError: Request failed with status code 400）？

A: 这通常是文件大小或格式问题，请检查：

- Excel文件大小不超过10MB
- JSON配置文件大小不超过500KB  
- 文件格式正确（Excel: .xlsx/.xls，配置: .json）
- 如果问题持续，请重启应用或清除浏览器缓存
- 建议使用系统默认数据进行首次测试

### Q: 网页无法打开？

A: 确保依赖包已正确安装，检查控制台错误信息

### Q: 优化算法执行失败？

A: 检查数据文件格式是否正确，确保Excel文件包含必要的距离信息

### Q: 出现"AttributeError: 'Individual' object has no attribute 'get'"错误？

A: 这个错误已在最新版本中修复。如果仍然遇到，请：

- 重启Web应用：`python start_webapp.py`
- 清除浏览器缓存后重新访问
- 确保使用的是最新版本的代码

### Q: 出现"TypeError: unsupported format string passed to NoneType"错误？

A: 这个错误也已在最新版本中修复。现在系统会：

- 安全地处理None值格式化
- 提供默认数值避免格式化错误
- 在所有数字显示位置使用安全格式化函数
- 如果仍遇到问题，请重启应用

### Q: 出现"TypeError: can only join an iterable"错误？

A: 这个错误也已在最新版本中修复。现在系统会：

- 安全地处理None值和非可迭代对象
- 自动过滤空值和无效数据
- 在生成报告时提供默认显示内容
- 如果仍遇到问题，请重启应用

### Q: 地图不显示？

A: 确保网络连接正常，地图使用OpenStreetMap服务

### Q: 如何使用智能距离计算功能？

A: 智能距离功能可以通过百度地图API获取真实距离：

1. **申请API密钥**：
   - 访问百度地图开放平台：<https://lbsyun.baidu.com/>
   - 注册并创建应用，获取API密钥(AK)

2. **配置API密钥**：
   - 在Web应用的"智能距离配置"区域
   - 输入百度地图API密钥
   - 点击"验证API"确认有效

3. **配置地点**：
   - 选择"使用默认地点"或"自定义地点列表"
   - 自定义时输入具体地点名称，系统自动解析坐标

4. **生成距离矩阵**：
   - 点击"生成智能距离矩阵"
   - 车辆距离：通过API获取真实行驶距离
   - 无人机距离：计算坐标直线距离

更多详情请参考：SMART_DISTANCE_GUIDE.md

### Q: API密钥验证失败？

A: 请检查以下几点：

- API密钥是否正确复制
- 应用状态是否启用
- IP白名单是否配置正确
- 配额是否充足

### Q: 结果下载失败？

A: 检查浏览器是否允许文件下载

## 🎯 使用流程

1. **启动应用** → 运行start_webapp.py或start_webapp.bat
2. **配置参数** → 在侧边栏调整车辆、无人机和算法参数
3. **配置数据源** → 可选择以下任一方式：
   - 上传自定义距离矩阵Excel文件
   - 使用智能距离计算功能（百度地图API）
   - 使用默认内置数据
4. **执行优化** → 点击"执行优化"按钮开始算法运行
5. **查看结果** → 在结果选项卡中查看配送方案、成本分析等
6. **下载报告** → 保存优化报告和JSON数据
